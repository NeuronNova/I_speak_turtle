# I Speak Turtle

> Learning Turtle faster than Turtle intended.

Not a course. Not a tutorial. Just a log of things I actually ran,
broke, and figured out while learning RDF and the Semantic Web through
Python and rdflib.

Everything here is Python-first. 

If your first instinct is to say "but what about Java / Scala / Rust / whatever",
respectfully, the Semantic Web was half-dead for a decade and Python is why
data scientists are the ones who brought it back. You're welcome.

## Why this repo exists
Most RDF learning starts with the W3C spec, a lecture on graph theory,
or a course that makes you watch three videos before you touch a terminal.
The community literally points beginners at the RDF 1.1 Primer as step one.
 
This repo starts with `g.add()` and works backwards from there.
 
The log grows when something clicks, goes quiet when something else takes over.
